# HinghwaDict
一款莆仙方言在线工具APP，以字词查询为基础，让莆仙方言更好融入到互联网的潮流之中。（目前正在不断完善中）

## Version

### V1.1.0(2022/5/27)
- 添加底部导航栏，包括首页和文章。
- 支持Markdown文章的查看。
- 首页可点击公告进行查看。

### V1.0.0(2022/5/25)
- 添加随机跳词和每日一词。
- 实现搜索功能，支持拼音和词语检索。
- 可查看词语详情页面。
